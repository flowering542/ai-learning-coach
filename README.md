# AI Learning Coach - 学习教练 HTTP 服务

## 快速开始

```bash
# 安装依赖
npm install

# 启动 HTTP 服务
./start.sh

# 或使用环境变量
COACH_PORT=3000 COACH_ADMIN_QQ_IDS=1933622876 node server.js
```

## API 文档

### POST /api/coach

主 API 端点，处理所有学习教练命令。

**请求体：**
```json
{
  "command": "/练习",
  "userId": "user_123",
  "platform": "feishu"
}
```

**响应：**
```json
{
  "result": "📝 练习题...",
  "data": { }
}
```

### GET /health

健康检查端点。

**响应：**
```json
{
  "status": "ok",
  "timestamp": "2026-03-06T..."
}
```

## 支持的命令

### 数字菜单
- `1` - 开始练习
- `2` - 查看进度
- `3` - 错题复习
- `4` - 薄弱分析
- `5` - 我的徽章
- `6` - 学习打卡
- `7` - 帮助说明

### 自然语言
- "练习"/"做题"/"继续" → 开始练习
- "进度"/"成绩" → 查看统计
- "错题"/"错了哪些" → 错题复习

### 传统命令
- `/练习` - 开始练习题
- `/进度` - 查看学习进度
- `/错题` - 查看错题
- `/分析` - 薄弱点分析
- `/徽章` - 成就徽章
- `/打卡` - 学习打卡

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| COACH_PORT | 3000 | HTTP 服务端口 |
| COACH_ADMIN_QQ_IDS | - | 管理员 QQ 号，逗号分隔 |
| COACH_DATA_DIR | ./data | 数据存储目录 |

## 特性

- ✅ HTTP Keep-Alive 长连接
- ✅ 毫秒级响应
- ✅ 多用户数据隔离
- ✅ 300 道真题库
- ✅ 自动出题（答完即出下一题）
- ✅ 错题记录与复习
- ✅ 学习进度统计
- ✅ 连续打卡追踪
