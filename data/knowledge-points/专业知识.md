## 一、安全献血

### 1. 献血类型

| 类型 | 定义 | 特点 |
|------|------|------|
| **家庭/替代献血** | 患者家属或社区成员提供血液 | 缓解供血不足，但增加患者压力，可能被迫献血 |
| **有偿/职业供血** | 为获得报酬而献血 | 健康风险高，可能频繁献血，WHO反对 |
| **志愿无偿献血** | 自愿捐献，不获取报酬 | 血液质量高，是安全献血的基础 |
| **固定献血者** | 至少献血3次，每年至少1次 | 低危献血者，传染病风险最小 |

### 2. 定向献血（不推荐）
- **定义**：献血者受特殊要求给指定患者献血
- **风险**：亲属间输血易发生**输血相关移植物抗宿主病**（死亡率>90%）
- **WHO要求**：血液必须存入血站，不能直接给指定患者输注

### 3. 我国献血规定
| 项目 | 标准 |
|------|------|
| 每次采血量 | 200ml，最多400ml |
| 采集间隔 | 不少于6个月 |
| 男性献血频率 | 3个月一次 |
| 女性献血频率 | 4个月一次 |

### 4. 特殊献血类型
| 类型 | 说明 |
|------|------|
| HLA同型献血 | 献血者与患者HLA相合 |
| 稀有血型献血 | Rh(D)阴性等稀有血型 |
| 缺乏抗原献血 | 缺乏IgA抗原，供IgA抗体阳性患者 |
| 试剂献血 | 红细胞含特殊抗原或血清含特殊抗体 |
| 特异抗体献血 | 含高效价特异抗体，可制备免疫球蛋白 |

### 5. 低危献血者
- **定义**：传播输血传染病危险性低的献血者
- **特点**：
  - 受过安全教育
  - 每次献血都经过化验
  - 了解危险行为后不再献血
  - 固定献血者通常被认为是低危献血者

## 二、血液采集

### 1. 献血者健康检查
- **必须项目**：
  - 体格检查
  - 血液检测（血型、血红蛋白、转氨酶等）
  - 传染病筛查（乙肝、丙肝、艾滋、梅毒）

### 2. 采血前准备
- 核对献血者身份
- 询问健康状况和危险行为
- 测量体重、血压、脉搏
- 血红蛋白检测（男≥120g/L，女≥115g/L）

### 3. 采血过程
- 采血环境符合卫生标准
- 使用一次性无菌器材
- 采血量为200-400ml
- 采血时间控制在10-15分钟

### 4. 献血后护理
- 压迫止血10-15分钟
- 观察30分钟无异常方可离开
- 告知注意事项（24小时内避免剧烈运动等）

## 三、血液成分制备

### 1. 全血
- **定义**：采集的血液未经分离，包含所有成分
- **保存**：2-6℃，CPDA-1保存液可保存35天
- **适应症**：急性大量失血、体外循环等

### 2. 红细胞制品
| 制品名称 | 特点 | 适应症 |
|----------|------|--------|
| 浓缩红细胞 | 去除大部分血浆 | 各种贫血 |
| 悬浮红细胞 | 加入添加剂 | 各种贫血，最常用 |
| 洗涤红细胞 | 去除血浆蛋白 | 对血浆蛋白过敏者 |
| 冰冻红细胞 | 甘油冷冻保存 | 稀有血型保存 |
| 辐照红细胞 | 经γ射线辐照 | 预防TA-GVHD |

### 3. 血小板制品
| 制品名称 | 制备方法 | 保存条件 |
|----------|----------|----------|
| 手工分离血小板 | 从全血中分离 | 22±2℃，震荡保存5天 |
| 机采血小板 | 血细胞分离机采集 | 同上 |

### 4. 血浆制品
| 制品名称 | 特点 | 适应症 |
|----------|------|--------|
| 新鲜冰冻血浆(FFP) | 含全部凝血因子 | 凝血因子缺乏 |
| 普通冰冻血浆 | FFP保存1年后 | 补充血浆蛋白 |
| 冷沉淀 | FFP融化后沉淀 | 纤维蛋白原缺乏 |

## 四、血液保存

### 1. 保存液
| 保存液 | 保存期 | 特点 |
|--------|--------|------|
| ACD | 21天 | 酸性枸橼酸盐葡萄糖 |
| CPD | 28天 | 枸橼酸盐磷酸盐葡萄糖 |
| CPDA-1 | 35天 | 加入腺嘌呤，最常用 |

### 2. 保存温度
| 血液成分 | 保存温度 | 保存期限 |
|----------|----------|----------|
| 全血/红细胞 | 2-6℃ | 35天(CPDA-1) |
| 血小板 | 20-24℃ | 5天 |
| 新鲜冰冻血浆 | ≤-18℃ | 1年 |
| 普通冰冻血浆 | ≤-18℃ | 4年 |
| 冷沉淀 | ≤-18℃ | 1年 |

### 3. 血液运输
- 运输温度与保存温度一致
- 使用专用血液运输箱
- 全程温度监控
- 运输时间尽量缩短

## 五、输血前检测

### 1. ABO血型鉴定
- **正定型**：用已知抗体检测红细胞抗原
- **反定型**：用已知抗原检测血清抗体
- **原则**：正反定型结果必须一致

### 2. Rh血型鉴定
- **必检项目**：D抗原
- **Rh阴性**：D抗原阴性
- **我国Rh阴性率**：汉族约0.4%

### 3. 交叉配血
| 方法 | 原理 | 特点 |
|------|------|------|
| 盐水介质法 | 检测IgM抗体 | 简单快速 |
| 抗人球蛋白法(Coombs) | 检测IgG抗体 | 最可靠 |
| 凝聚胺法 | 促进抗原抗体结合 | 灵敏度高 |
| 微柱凝胶法 | 凝胶过滤 | 标准化高 |

### 4. 抗体筛查
- **目的**：检测受血者血清中是否有不规则抗体
- **方法**：使用筛选细胞（含主要血型抗原）
- **意义**：预防溶血性输血反应

### 5. 传染病检测
| 检测项目 | 意义 |
|----------|------|
| 乙肝表面抗原(HBsAg) | 预防乙肝传播 |
| 丙肝抗体(抗-HCV) | 预防丙肝传播 |
| 艾滋病毒抗体(抗-HIV1/2) | 预防艾滋病传播 |
| 梅毒螺旋体抗体(抗-TP) | 预防梅毒传播 |
| 谷丙转氨酶(ALT) | 肝功能指标 |
