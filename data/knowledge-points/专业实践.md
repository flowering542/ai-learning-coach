## 一、临床输血操作

### 1. 输血原则
- **能不输就不输，能少输不多输**
- **缺什么补什么**
- 合理利用血液资源

### 2. 输血前准备与申请

#### 医师职责
- 严格掌握输血适应证
- 正确应用临床输血技术（成分输血、自体输血）
- 向患者/家属说明：
  - 输血不良反应（过敏、发热）
  - 经血传播疾病（HIV、HBV、梅毒、寄生虫等）
- 签署《输血治疗同意书》

#### 必做检查
| 检查项目 | 意义 |
|----------|------|
| ABO血型 | 血型鉴定 |
| Rh血型 | 血型鉴定 |
| Hb、HCT | 贫血程度 |
| 血小板 | 凝血功能 |
| 不规则抗体筛查 | 预防溶血反应 |
| ALT | 肝功能 |
| 乙肝两对半 | 乙肝筛查 |
| 抗-HCV | 丙肝筛查 |
| 抗-HIV1/2 | 艾滋筛查 |
| RPR | 梅毒筛查 |

#### 特殊规定
| 情况 | 处理 |
|------|------|
| 用血量>2000ml | 科主任签字后报医务科，急诊除外（事后补办） |
| 无家属签字紧急输血 | 报医务科/分管领导同意、备案 |
| Rh(D)阴性/稀有血型 | 自身输血、同型输血或配合型输血 |
| 亲友互助献血 | 到血站无偿献血，血站负责检测和调配 |

## 二、血型鉴定操作

### 1. 凝聚胺法ABO血型鉴定

#### 操作步骤
1. 取试管2支，标记"A"管（加A型血清/抗B）、"B"管（加B型血清/抗A）
2. 各加已知血清2滴，再加待检者3-5%血球1滴
3. 各加LIM（低离子液）0.6ml，混匀
4. 各加2滴Polybrene Solution（凝聚胺液），混匀
5. 3400rpm离心10秒（或1000转/分，1分钟）
6. 倒掉上清，留约0.1ml液体
7. 轻摇试管，目测红细胞有无凝集
8. 加2滴Resuspending（复悬液），轻混，判断结果

#### 结果判断
| A管 | B管 | 血型 |
|-----|-----|------|
| 凝集散开 | 凝集不散开 | A型 |
| 凝集不散开 | 凝集散开 | B型 |
| 均凝集不散开 | 均凝集不散开 | AB型 |
| 均凝集散开 | 均凝集散开 | O型 |

**注意**：凝集管不能排除免疫性抗体的存在

### 2. 血型鉴定要求
- 结果必须在显微镜下观察证实
- 见《全国检验操作规程》第二版 P89-95

## 三、交叉配血操作

### 1. 血样要求
| 项目 | 要求 |
|------|------|
| 血样时效 | 输血前3天以内 |
| 血量 | 不少于2ml |
| 多次配血 | 每增加一次，血量增加1ml |
| 特殊注意 | 右旋糖酐干扰配血，输注前采集 |

### 2. 配血前核对
- 《输血申请单》
- 受血者和供血者血样
- 复查ABO血型（正反定型）
- 常规检查Rh(D)血型（紧急输血可除外）

### 3. 抗体筛选
**必须做的情况**：
- 交叉配血不合
- 有输血史
- 有妊娠史
- 短期内需要多次输血

### 4. 血样保存
- 交叉配血后，受血者和供血者血样密封
- 2-6℃保存至少7天

### 5. 配血方法选择
| 方法 | 适用情况 |
|------|----------|
| 盐水介质配血法 | 初次受血或不规则抗体筛选阴性 |
| 凝聚胺配血法 | 盐水配血不合、有输血/妊娠史、多次输血 |

### 6. 凝聚胺交叉配血操作

#### 试剂准备
- 3-5%血球悬液：1ml生理盐水 + 30-50μl压积红细胞

#### 操作步骤
1. 取试管2支，标记主侧、次侧
   - 主侧：病人血清2滴 + 供血者血球1滴
   - 次侧：献血者血清2滴 + 病人血球1滴
2. 各加LIM 0.6ml，混匀，室温静置1分钟
3. 各加2滴Polybrene Solution，静置15秒
4. 3400rpm离心10秒，倒掉上清，留0.1ml
5. 轻摇试管，目测凝集
6. 加2滴Resuspending，轻混，观察结果
7. 显微镜下确认

#### 结果判断
- **凝集不散开**：抗体结合的特异性反应，配血不相容
- **凝集散开**：Polybrene引起的非特异性凝集，配血相容

#### 报告方式
- **同型配血**："主、次管交叉配血无凝集、无溶血反应"
- **异型配血**（不主张）："主管无凝集、无溶血；次管有凝集、无溶血"

### 7. 抗体筛选操作（凝聚胺法）
1. 取2支试管，各加病人血清2滴 + 3-5%筛检血球1滴
2. 各加LIM 0.6ml，混匀
3. 各加2滴Polybrene Solution，混匀
4. 3400rpm离心10秒，倒掉上清，留0.1ml
5. 轻摇试管，目测凝集
6. 加2滴Resuspending，轻混，观察结果

#### 结果判断
- **凝集散开**：抗体筛查阴性
- **凝集不散开**：抗体筛查阳性，需进一步鉴定

### 8. 需要交叉配血的血液成分
- 全血
- 浓缩红细胞
- 红细胞悬液
- 洗涤红细胞
- 冰冻红细胞
- 浓缩白细胞
- 手工分离浓缩血小板

**注意**：机器单采浓缩血小板应ABO同型输注

## 四、血液入库、核对、储存

### 1. 入库核对内容
- 运输条件
- 物理外观
- 血袋封闭及包装
- 标签内容（供血机构、姓名/条码、血型、品种、容量、采血日期、制备日期、血袋编号、储存条件）

### 2. 储存要求
| 项目 | 要求 |
|------|------|
| 分类储存 | 按A、B、O、AB血型分储，有明显标识 |
| 冰箱管理 | 严禁存放其他物品 |
| 消毒 | 每周消毒一次 |
| 空气培养 | 每月一次，细菌<8CFU/10分钟或<200CFU/m³ |

### 3. 温度监控
- 温度自动控制记录和报警装置
- 报警时立即检查原因、解决并记录

## 五、血液出库（发血）

### 1. 取血规定
- **通知方式**：配血合格后电话通知用血科室
- **取血人员**：医护人员（禁止患者/家属取血）
- **特殊情况**：急诊时可由配血者/非用血科室人员送血

### 2. 发血核对（九项核查）
双方共同查对：
1. 患者姓名
2. 性别
3. 病案号
4. 门急诊/病室
5. 床号
6. 血型
7. 血液有效期
8. 配血试验结果
9. 保存血的外观

**确认无误后双方签字**

### 3. 血液发出后
- **不得退回**
- 要求临床医师严格把握

### 4. 不得发出的血袋
| 情况 | 说明 |
|------|------|
| 标签破损、字迹不清 | 无法识别 |
| 血袋破损、漏血 | 污染风险 |
| 血液中有明显凝块 | 质量异常 |
| 血浆呈乳糜状或暗灰色 | 质量异常 |
