## 一、中华人民共和国献血法

### 1. 基本规定
- **立法目的**：保证医疗临床用血需要和安全，保障献血者和用血者健康
- **献血制度**：国家实行无偿献血制度
- **献血年龄**：18-55周岁健康公民自愿献血
- **领导机构**：地方各级人民政府领导，卫生行政部门监督管理

### 2. 血站管理
| 项目 | 规定 |
|------|------|
| 性质 | 不以营利为目的的公益性组织 |
| 审批 | 国务院或省级卫生行政部门批准 |
| 健康检查 | 免费进行必要的健康检查 |
| 采血量 | 一般200ml，最多400ml |
| 采集间隔 | 不少于6个月 |
| 禁止行为 | 超量、频繁采集血液 |

### 3. 血液管理
- **血液用途**：无偿献血的血液必须用于临床，不得买卖
- **血液检测**：血站必须对采集的血液进行检测
- **不合格血液**：不得向医疗机构提供
- **临床核查**：医疗机构必须核查，不符合标准的不得使用

### 4. 费用规定
- **临床用血费用**：只交付采集、储存、分离、检验等费用
- **无偿献血者**：临床用血时免交上述费用
- **直系亲属**：可按省级规定免交或减交

### 5. 应急用血
- **自身储血**：提倡择期手术患者自身储血
- **互助献血**：动员家庭、亲友、单位互助献血
- **临时采集**：医疗机构可临时采集，但须确保安全

### 6. 法律责任
| 违法行为 | 处罚 |
|----------|------|
| 非法采集血液 | 取缔，没收违法所得，可并处10万元以下罚款 |
| 出售无偿献血血液 | 同上，构成犯罪的追究刑事责任 |
| 血站违反操作规程 | 责令改正，造成损害依法赔偿，行政处分 |
| 提供不合格血液 | 责令改正，情节严重限期整顿，追究责任 |
| 使用不合格血液 | 责令改正，造成损害依法赔偿，行政处分 |

## 二、医疗机构临床用血管理办法

### 1. 基本原则
- **用血范围**：全血和成分血，不得使用原料血浆（科研项目除外）
- **用血原则**：合理、科学，不得浪费和滥用
- **供血来源**：县级以上人民政府卫生行政部门指定的血站

### 2. 组织机构
| 机构 | 职责 |
|------|------|
| 临床输血管理委员会 | 规范管理和技术指导，教育培训 |
| 输血科（血库） | 计划申报、储存血液、检查制度执行、参与诊疗 |

### 3. 血液收领核查
**血袋包装核查内容**：
1. 血站名称及许可证号
2. 献血者姓名（或条形码）、血型
3. 血液品种
4. 采血日期及时间
5. 有效期及时间
6. 血袋编号（或条形码）
7. 储存条件

### 4. 血液储存
| 血液品种 | 储存温度 |
|----------|----------|
| 全血、红细胞、代浆血 | 2-6℃ |
| 血小板 | 20-24℃（6小时内输注） |

- 储血设施应保证完好
- 24小时监测记录温度
- 储血环境符合卫生学标准

### 5. 输血适应症
- **血红蛋白**：<100g/L
- **血球压积**：<30%

### 6. 输血审批
| 用血量 | 审批要求 |
|--------|----------|
| 常规 | 经治医师申报，上级医师核准签字 |
| >2000ml | 输血科医师会诊，科室主任签名，医务处批准 |
| 急诊 | 事后补办手续 |

### 7. 输血前告知
- 输血目的
- 可能发生的输血反应
- 经血液途径感染疾病的可能性
- 医患双方签署用血志愿书或输血治疗同意书

### 8. 发血与输血
- **领血**：专人持配血单领取，认真核查
- **发血**：检查领血单填写项目，合格后方可发血
- **输血前**：核对血型、品种、规格、有效期
- **记录**：输血情况详细记入病历

### 9. 自身输血与互助献血
- **自身储血**：平诊和择期手术患者，由在治医疗机构采集
- **亲友献血**：由血站采集、检测，负责调配合格血液
- **考核**：作为评价医生工作业绩的重要内容

### 10. 成分输血
- **推行**：医疗机构应积极推行血液成分输血
- **比例**：达到卫生部规定要求
- **制备**：省级以上卫生行政部门批准的血站负责

### 11. 应急临时采集条件
必须同时符合：
1. 边远地区，所在地无血站（或中心血库）
2. 危及病人生命，急需输血，其他措施不能替代
3. 具备交叉配血及快速诊断方法（乙肝、丙肝、艾滋病毒检测）
